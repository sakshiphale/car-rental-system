# 🚗 Car Rental System

A simple and interactive **Java-based Car Rental System** that showcases Object-Oriented Programming (OOP) principles through a console-based app.

---

### 🔹 Features

* 🚘 View available cars
* 📦 Rent and return vehicles
* 🧾 Display rental info

---

### 🛠️ Tech Stack

* Java (Core)
* File I/O
